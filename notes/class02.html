CSS Intro

CSS - History

• CSS => Cascading Style Sheets

• First proposed in 1994 by Håkon Lie ("How Con Lee") - a colleague of Tim Berners-Lee

• Stylesheet language used to describe HTML markup

• Allows separation of document content (HTML), from document presentation/styling

⇒ "Separation of concerns"

CSS - Syntax

• This CSS rule turns the text color of every paragraph, <p>, black

CSS - Syntax (ctn)

• One selector can have multiple declarations

• Common for each declaration to have its own line

CSS Color Property

CSS Color Property

• Color values can be specified using:

⇒ color keyword: red

⇒ rgb: rgb(255, 0, 0)

⇒ hex code: #FF0000

• Can also use: RGBA, HSL, HSLA

CSS Color: Keyword Value

	span { color: red; }

• Simplest; least-descriptive

• red, green, blue, etc.

    ⇒ tomato

    ⇒ deeppink

    ⇒ chartreuse

More colors...

CSS Color: RGB Value

	span { color: rgb(255, 0, 0); }

• More complicated; more powerful!

• First value is red; second is green; third is blue

• Each value in a range 0 - 255

• Values closer together are gray or muted:

rgb(200,200,200) / rgb(155,155,155) / rgb(075,075,075)

CSS Color: Hexadecimal Value

	p { color: #463DB3 };
  • How does it work?
    • Great blog post "The Code Side of Color": http://www.smashingmagazine.com/2012/10/the-code-side-of-color/
  • Extracting Hex Values: from code, or from image using Photoshop / software tool.
  • When you're the designer (pick a palette):
    • Adobe Color: https://color.adobe.com
Applying Color

• color

⇒ Used to apply color to text


	p {
	  color: rgb(255, 0, 0);
	}

• background

⇒ Used to apply a background color (or image) to elements:


	p {
		color: rgb(255, 0, 0);
		background: rgb(0, 255, 0);
	}

Fonts + Text

CSS Font: Various Properties

• font-weight

⇒ Sets the "thickness" of the font

⇒ normal and bold; values 100 - 900

• font-style

⇒ Sets a "slant" to the font

⇒ normal and italic

• font-size

⇒ Sets the size of the font

⇒ Pixel values (12px); percentage (75%); em's (1em)

CSS Font: Font-Family Property

  p { font-family: 'Comic Sans MS', helvetica, serif };

• font-family

⇒ Applies a particular font to the element

⇒ Accepts multiple values

⇒ Should always end with: serif; sans-serf; monospace;

Working with CSS

Working with CSS

Three ways to apply CSS to HTML:

1) Inline CSS

2) Embed a <style> tag inside the <head>

3) Link to an external file using the <link> tag

Inline CSS

	<p style="color: blue; font-size: 14px;">
		Keanu Reeves is Johnny Utah
	</p>

	<p style="color: blue; text-decoration: underline;">
		Or is Johnny Utah Keanu Reeves?
	</p>

• Not a good practice

• Does not separate our concerns (HTML/CSS)

• Makes debugging difficult

• ONLY USE IF WRITING HTML EMAILS

• You will make enemies

Do NOT Use Inline CSS!

<head> embed

	<!DOCTYPE html>
	<html>
	<head>
		<title>Super Awesome Website</title>
		<style type="text/css">
			h1 {color: red;}
			p {color: blue; font-size: 14px}
		</style>
	</head>
	<body>
		...

• Better than inline styles

• Bloats HTML pages

• Can make debugging large applications difficult

External CSS

 <!DOCTYPE html>
 <html>
 <head>
 	 <title>Super Awesome Website</title>
	 <link rel="stylesheet" type="text/css" href="style.css">
 </head>
 <body>
	...

• Like <title>, links to CSS files go in <head>

• Truly separates HTML from CSS

• Easiest to debug; just go to your CSS file!

External CSS

     <link rel="stylesheet" type="text/css" href="style.css">

• rel

⇒ Specifies the relationship between the current document and the linked document

• type

⇒ Specifies the media type of the linked document

• href

⇒ Specifies the location of the linked document
