Toggling Element Visibility

jQuery Hide and Show

.hide()

⇒ Hides elements by setting their inline style to display: none;


	$('p').hide();

.show()

⇒ Reveals elements by setting their inline style to display: block

⇒ If the element was originally inline, then display: inline will be used


	$('p').show();
jQuery Hide and Show

• Change the duration by passing an argument:

⇒ String:


	$('p').hide('slow')

	$('p').show('fast')

⇒ Number:


	$('p').hide(700)

	$('p').show(300)
jQuery Hide and Show

• jQuery provides alternatives to .hide() and .show():

.fadeIn()

.fadeOut()

.slideDown()

.slideUp()

.slideToggle()

• Each takes an optional duration argument:


	$('p').fadeIn('fast')

	$('p').slideUp(900)

Manipulating Element Attributes

Manipulating Element Attributes

.attr('attribute-name')

⇒ Gets the value of the matched attribute name


	

	$('img').attr('src')
	// returns 'keanu.jpg'

.attr('attribute-name', 'attribute-value')

⇒ Changes the value of 'attribute-name' to 'attribute value'


	$('img').attr('src', 'gary-busey.jpg')
Manipulating Element Attributes

• Assuming the following HTML:


	<a href="http://google.com">Google
	<img id="logo" src="smiley.png">

• Get attribute .attr('attribute-name'):


	$('a').attr('href')

	// returns 'http://google.com'



	$('#logo').attr('id')

	// returns 'logo'
Manipulating Element Attributes

• Assuming the following HTML:


	<img id="logo" src="smiley.png">

• Set attribute .attr('attribute-name', 'attribute-value'):


	$('#logo').attr('src', 'frowney.png')

• Turns the above HTML into:


	<img id="logo" src="frowney.png">

$(this)

• $(this) allows us to target specific elements that invoked (or called) a function

• Refactor previous CodePen using $(this):


	$('span').click(function () {
	  $(this).css('color', 'red')
	})

• On the click event, only the <span> that invoked the function will turn red

$(this)

• In JS, functions are called (invoked) by an object

• Whatever object called the function can be accessed inside the function using this

• this is said to be the "context" of the function (refers to whatever object invoked/called it)

• We can "wrap" this in jQuery: $(this)

• Remember, DOM nodes are objects
