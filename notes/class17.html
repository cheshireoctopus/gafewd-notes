HTML5 Video: Other Attributes

• controls

⇒ Allows user to control volume, seeking, pause/play

• autoplay

⇒ Video will begin playing automatically (usually before loading has completed)

• loop

⇒ Loops video

• poster="path/to/src.jpg"

⇒ Adds background image

HTML5 Video: Pros/Cons

Pros:

⇒ Standardized <video> tag

⇒ Video-file neutral (MP4, Ogg, WebM)

⇒ No more Flash!

Cons:

⇒ Only works on modern browsers (no < IE9)

⇒ Not all browsers support same video-file types

jQuery Plugins

jQuery Plugins

• Plugins (or libraries) are pre-written pieces of JavaScript

• Built to solve a particular issue:

⇒ Fullscreen video player

⇒ Image carousel

⇒ Page scrolling

• To use, just add the plugin script below jQuery:


	<script> src="jquery.js"></script>
	<script> src="my-super-plugin.js"></script>
jQuery Plugins: What to look for?

• There are 1000's of plugins; be judicious:

⇒ Recently updated

⇒ Well-documented

⇒ The more contributors the better

⇒ Open source

⇒ Hosted on GitHub

jQuery Plugins: Tips for using

1) Read the documentation

2) Seriously, read the documentation

3) Look for, and copy, examples

4) Start simply, then add complexity

5) Search for tutorials/examples if you get stuck

6) Open a GitHub issue if you think you found a bug
