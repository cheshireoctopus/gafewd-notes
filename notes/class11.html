Conditional Flows JavaScript Syntax:


	var x = 10;

	if (x > 5) {
	  alert('x is larger than 5!');
	} else {
	  alert('x is smaller than 5!');
	}

• Denoted by the if and else keywords

• Begins with a condition: (x > 5)

• If the condition is true, the statment(s) in the "if block" will run

• If the condition is false, the statment(s) in the "else block" will run

Conditional Statements: Multiple Conditions

• Often, programs will evaluate more than one condition

• In JS, use the else if for multiple conditions:


	var x = 10;

	if (x != 10) {
		alert('beer');
	} else if (x == 10) {
		alert('coffee');
	} else {
		alert('water');
	}

Chaining Functions

Chaining Functions

How can we improve this code:


	function doSomething () {
	  $('p').css('background-color', 'blue')
	  $('p').text('Some new text to be added')
	  $('p').slideDown()
	}

Function chaining:


	function doSomething () {
	  $('p')
	    .css('background-color', 'blue')
	    .text('Some new text to be added')
	    	.slideDown()
	}
Chaining Functions

• Perform multiple effects/actions one after another

• Only works when called on the SAME object:


	$('p').text('hello').css('color', 'red').slideDown()

• Some chained commands are queued; commands further down the chain will wait until previous commands have finished before executing


	$('p').slideDown(500).slideUp(500).slideDown(500)
