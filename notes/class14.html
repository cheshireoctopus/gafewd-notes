Cash Register Review

box-sizing: border-box;

box-sizing: border-box;

Q: What is the width of the following element? (Hint: it's not 100px...)


	.box {
	  height: 100px;
	  width: 100px;
	  border: 1px solid black;
	  padding: 20px;
	}

A: 142px

100px in width

40px of padding (20px on left and right)

2px of border (1px on left and right)

box-sizing: border-box;

• By default, the width of an element is the sum of:

⇒ width (content)

⇒ padding

⇒ border

• A bit counter-intuitive as the set width of an element increases as you add padding and border

• CSS3 introduced box-sizing to address the confusion

box-sizing: border-box;

• Apply the box-sizing: border-box; rule to the previous CodePen


	.box {
	  height: 100px;
	  width: 100px;
	  border: 1px solid black;
	  padding: 20px;
	  box-sizing: border-box;
	}

• Right-click to inspect and examine the element's overall width

Fluid Layout

Fixed Layout

• So far, our layouts have been fixed (eg 960px)

• Container (wrapper) element has a fixed-width

• Content will render the same across screen sizes

Fluid Layout

• Fluid (aka liquid) layouts use %-based widths (eg width: 75%)

• Content will adapt to the screen size:

• 75% of 960px => 720px

• 75% of 375px => 280px

Media Queries

Media Queries

A Definition:

A media query consists of a media type and at least one expression that limits the style sheets' scope by using media features, such as width, height, and color. Media queries, added in CSS3, let the presentation of content be tailored to a specific range of output devices without having to change the content itself.

- MDN

Media Queries

Another Definition:

Media queries allow you to create CSS styles based on a variety of rules, like screen size.

For Example:

• When width > 500px color text red

• When width < 500px color text blue

• CodePen Example

Media Queries: Methods

1. Within CSS Stylesheet:


	@media screen and (min-width: 1024px) {
	  /* Styles go here */
	}

2. Using a <link> tag:


  <link rel="stylesheet " media="screen and (min-width: 1024px)" href="css/1024only.css">
Media Queries: Browser-Width

Within CSS Stylesheet:


	@media screen and (min-width: 1024px) {
	  color: red;
	}

Media Type: screen

Only target devices with a viewable screen

Rule: (mid-width: 1024px)

Only target windows larger than, or equal to, 1024px

Media Queries: Media Types

all: Suitable for all devices

handheld: Intended for handheld devices

print: Intended for paged material and for documents viewed on screen in print preview mode

screen: Intended primarily for color computer screens

tv: Intended for television-type devices

- W3.org

Media Queries: Media Rules

width: viewport width

height: viewport height

aspect-ratio: width-to-height aspect ratio of the viewport

orientation: orientation of the viewport

- MDC

Media Queries: Browser-Width

• min-width: ___ px

Targets widths greater than or equal to __ px wide


	@media screen and (min-width: 1000px) { ... }

• max-width: ___ px

Targets widths less than or equal to __ px wide


	@media screen and (max-width: 1000px) { ... }
Media Queries: Additional Examples

Don't have to specify a media type:


	@media (min-width: 700px) { ... }

Multiple rules:


	@media (min-width: 320px) and (max-width: 480px) { ... }

• These styles will only apply on devices between 320 and 480 pixels
